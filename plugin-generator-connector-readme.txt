Connect your WordPress site to the Plugin Generator app for direct plugin deployment.

== Description ==

The Plugin Generator Connector plugin allows you to connect your WordPress site to the Plugin Generator app, enabling direct deployment of generated plugins to your site.

**Features:**

* Secure API key generation for authentication
* Direct plugin installation and activation
* Simple setup process

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-generator-connector` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Plugin Generator to view your API key.
4. Use this API key to connect your WordPress site to the Plugin Generator app.

== Frequently Asked Questions ==

= Is this plugin secure? =

Yes, the plugin uses a unique API key for authentication and requires proper WordPress permissions to install plugins.

= Can I revoke access? =

Yes, you can regenerate your API key at any time to revoke access from previously connected applications.
